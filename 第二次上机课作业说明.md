## 四、技术实现要点

### 1. 依赖注入
所有组件都使用构造函数注入：
```typescript
constructor(private movieService: MovieService) {}
```

### 2. Observable 模式
```typescript
// Service 中
private moviesSubject = new BehaviorSubject<Movie[]>(this.movies);
public movies$ = this.moviesSubject.asObservable();

// 组件中
movies$ = this.movieService.movies$;

// 模板中
*ngFor="let movie of movies$ | async"
```

### 3. 数据流架构
```
MovieService (数据源)
    ↓
BehaviorSubject (状态管理)
    ↓
Observable (movies$)
    ↓
Components (订阅者)
    ↓
Templates (async 管道)
```

### 4. 日志记录
所有数据操作都通过 LoggerService 记录：
```typescript
this.logger.log(`添加新电影: ${movie.title} (ID: ${newId})`);
```
